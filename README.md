## Angular File Reader

### Connect to your project
* Add to your HTML file
```html
    <script src="../riki34.angular.file_reader.js"></script>
```

* Add dependency to your module
```javascript
    angular.module('your_module', [ 'riki34.angular.file_reader' ]);
```
